Preview the live website at **https://magnaye.online**.

## Use as a Template

1. Create a new repository on GitHub.
2. Clone your repository:

```bash
git clone https://github.com/your-username/your-repository.git
cd your-repository
```

3. Generate the portfolio template:

```bash
npx degit ratatatatcode/portfolio .
```

4. Install dependencies and start the development server:

```bash
npm install
npm run dev
```

5. Open **http://localhost:3000** in your browser and start customizing the portfolio.
